# The Beya
A Private Tech Oasis for edge AI inference, network engineering, and self-hosted services.

The Beya is my personal Homelab Infrastructure(Tech Oasis)—a segmented, secure network environment built for edge AI inference, self-hosted services, and continuous learning. Named after the training halls of sumo wrestlers, it reflects the same philosophy: deliberate practice, precise architecture, constant refinement.

Why I went this this set-up:

This was just a start to get a hang of the process of building a Network Infrastructure, deploying local AI models, and Network Segmentation. 
In the future I plan on upgrading to enterprise-grade/prosumer networking equipment to get more hands on experience tailored to small businesses and larger infrastructures. 
I am learning as I go but have already knowledge significant in Linux, zero-trust mesh networking, and also segementations.
